# 🌐 网页版 Gemini 零代码使用指南
# No-Code Guide for Gemini Web

> 本指南专为**法务、律师和业务人员**设计。无需配置 Python 编程环境或 API，只需通过浏览器即可直接体验 **Contract Reviewer Agent v2.0** 的完全体。

---

## 🏆 方法一：使用 Gemini Advanced 创建专属“Gem”（强烈推荐）

如果您拥有 **Gemini Advanced** 订阅（包含 Gemini 1.5 Pro 超大 200 万 token 上下文，非常适合处理长文本合同），您可以将 v2.0 永久固化为您的私人 AI 法务总监。

### 🛠️ 设置步骤（一劳永逸）：
1. 打开网页版 [Gemini](https://gemini.google.com/)，点击左侧边栏底部的 **“Gem 管理” (Gem Manager)**。
2. 点击 **“新建 Gem” (Create an Expert)**。
3. **名称 (Name)**：填写 `高级法务审查 Agent v2.0`。
4. **系统指令 (Instructions)**：打开本仓库的 [`skills/senior-legal-contract-reviewer-v2/SKILL.md`](../skills/senior-legal-contract-reviewer-v2/SKILL.md) 文件，将里面的**全部文字**完整复制并粘贴到指令框中。
5. 点击**保存**。

### 🚀 日常使用方式：
1. 在左侧边栏找到并选中刚创建的该 Gem。
2. 上传您的合同（支持 Word、PDF、或直接粘贴文本/截图）。
3. 附带上一句话设定您的参数，例如：
   > *"我是这份租赁合同的乙方（承租人），坐标中国大陆，总租金 800 万。请深度审核。"*
4. 随后，Agent 将自动分发给 6 个虚拟后台专家（合规、量化、谈判、生命周期、商业、质检），并在数十秒内由 ORCHESTRATOR 为您输出一份包含**五维风险评分**、**质量审计矩阵**及**条款修改建议**的**七章结构化正式风控报告**。

## 🆓 方法二：在普通免费版 Gemini 中直接加载

如果您使用的是免费版 Gemini（即基础版 Gemini Flash），无法创建自定义 Gem，您仍可以通过“手动喂入系统提示词”的方式单次激活它。

### 🛠️ 每次使用的步骤：
1. 打开网页版 [Gemini](https://gemini.google.com/)，开启一个**新的空白对话**。
2. 复制[`SKILL.md`](../skills/senior-legal-contract-reviewer-v2/SKILL.md) 全文。
3. 在发送框中加上如下激活语并发送：
   > *"请你立刻吸收并套用以下『高级法务合同审核智能体 v2.0』的全部系统架构、工作流和禁令。收到请仅回复『v2.0 多智能体架构已加载，请提供合同文本及您的立场』，不要展开其他内容。以下为系统规则：\n [粘贴 SKILL.md 全文]"*
4. 等待 Gemini 确认加载完毕（回复您上述指令）。
5. 将具体的合同文本发给它，并指明您的立场（甲方/乙方、金额等）。

### ⚠️ 免费版注意事项：
- 免费版 Gemini 的上下文记忆窗口较短。如果合同页数极多（如超过 30 页），在经过多次对话追问后，AI 可能会遗忘最开始设定的针对“违约金 130% 实务调减”的特殊计算规则。
- 建议一次性把整份合同发完，并要求它一次性输出完整报告。如果发现它有“偏离人设”或出现“法条幻觉”的倾向，可以再次把 `SKILL.md` 喂给它以重置状态。
