## Quick Usage

### 1) Load system prompt
- Use the full content of `SKILL.md` as the system instruction.

### 2) Provide task input
- Contract snippet or full text
- Party role
- Contract amount/range
- Review focus (payment, IP, default, dispute, etc.)

### 3) Expect structured output
- Risk level
- Identified vulnerability
- Expected loss estimation
- Legal citations (+ confidence degrade warning when needed)
- Plan B replacement clause

## Minimal Example

```json
{
  "risk_level": "HIGH",
  "identified_vulnerability": "Liability cap may block full recovery in fundamental breach scenarios.",
  "expected_loss_estimation": {
    "amount_range": "100W - 300W RMB",
    "calculation_logic": "EL = probability × median loss; apply judicial reduction logic when penalty is excessive."
  },
  "legal_citations": [
    {"law_name": "中华人民共和国民法典"}
  ],
  "citation_verified": false,
  "confidence_degrade": "Memory-based citation; lawyer verification required.",
  "defense_plan_b": "Add explicit exception for fundamental breach, confidentiality breach, and IP infringement."
}
```
